Intro

Installable via npm install --save aegir, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save aegir

You can then require() aegir as normal:

const aegir = require('aegir')

In the Browser

Aegir should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of aegir, including index.js and index.min.js, are included in the /dist folder. Aegir can also be found on unkpkg.com under

  • https://unpkg.com/aegir/dist/index.min.js
  • https://unpkg.com/aegir/dist/index.js
